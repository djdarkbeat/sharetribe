Overview

ServiceClient is a generalized way to call external services from the Rails application

It uses an Interceptor pattern, which makes it very easy to tailor a client that is suitable for different needs. It's easy to pick only the middleware that is needed and it's also easy to extend the client by writing a new middleware.
